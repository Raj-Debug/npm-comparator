# npm compare

A side-by-side npm package comparison tool built with React + Vite.

## Features
- Compare two npm packages at once
- Shows description, latest version, license, last modified
- Weekly and all-time download counts
- Responsive design

## Tech Stack
- React
- Vite
- Tailwind CSS
- npm Registry API
- npm Downloads API

## Run Locally
```bash
npm install
npm run dev
```
